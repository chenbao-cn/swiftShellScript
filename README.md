# swiftShellScript

用 Swift 语言来写 shell 脚本。
利用 Xcode 或其他代码编辑器的 "自动补齐" 功能，
让写 shell 脚本 不再那么痛苦。



## 期待用法:
```swift
git.commit.all.message("修复了一百个 Bug.")

git.log
// commit 54f1f42d2b96a3e1cb62gbeb8e4d0d4e96fc9ed4
// Author: ChenBao <email@email.com>
// Date:   2022-02-20 10:09:24 +0800
// 
//     修复了一百个 Bug.
// 
```
